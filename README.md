# RealPrice
RealPrice — реальная цена

Приложение на Flask для сравнения цен на продукты в разных магазинах. Помогает найти самую выгодную упаковку, автоматически пересчитывая стоимость в единую единицу измерения (например, цену за килограмм или за 100 граммов).

## Возможности

- 📱 Адаптивный интерфейс (работает на телефоне и ПК)
- 🔄 Автоматический пересчёт цены за килограмм/литр/100 г
- 📊 Годовой отчёт с графиками изменения цен
- 🏪 Управление списком магазинов (с цветовой маркировкой)
- 📏 Гибкая настройка единиц измерения и правил конвертации
- 🛒 Добавление/редактирование списка продуктов
- 🔒 Парольная защита

## Технологии

- Python (Flask)
- SQLite
- Chart.js
- Bootstrap (через CDN)
- venv

## Запуск

```bash
pip install flask
python app.py
```

Откройте http://localhost:5001
Пароль 0000

## Как работает
Выберите магазин, товар, укажите вес упаковки и цену
Приложение само пересчитает цену за стандартную единицу
Сравнивайте топ-5 самых дешёвых позиций
Следите за динамикой цен в отчёте за 12 месяцев
